## 📦 Requirements
Install dependencies into a virtual environment (Python 3.9 recommended):

```bash
python -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

```text
numpy
matplotlib
jupyter
sympy
```
